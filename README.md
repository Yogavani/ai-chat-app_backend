# AI Chat App

A full-stack chat application built with React Native and Fastify.

## Tech Stack

Frontend
- React Native
- TypeScript
- React Navigation

Backend
- Fastify
- MySQL
- JWT Authentication

## Features

- User registration
- Login with JWT
- Persistent login using AsyncStorage
- User list
- Chat screen
- Send and receive messages

## Backend Setup

cd ai-chat-backend
npm install
npm run dev

## Mobile Setup

cd AIChatApp
npm install
npx react-native run-android
